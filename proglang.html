Terms Definition
Machine language Machine language is the lowest level of programming language that directly corresponds to the instructions executed by a computer's hardware.
Debugging Debugging is the process of finding and fixing errors or bugs in the source code of any software.
Assembler An assembler is a type of computer program that takes in basic instructions and converts them into a pattern of bits that the computer's processor can use to perform basic operations.
Assembly language Assembly language uses mnemonics, which are symbolic representations of specific code -- known as operation code -- meant to be intuitive to programmers. h
Translator a computer program that converts the programming instructions written in human convenient form into machine language codes that the computers understand and process.
Complier a computer program that translates computer code written in one programming language (the source language) into another language (the target language).
Interpreter software that examines a computer program one instruction at a time and calls on code to execute the machine operations required by that instruction.
Cross-platform software a type of application / program / software that works on various operating systems or devices, which are often called platforms.
Natural language a machine learning technology that gives computers the ability to interpret, manipulate, and comprehend human language.
Eve Jackson
Formal language an abstract language used to express definitions, instructions and logic.
Machine independence A term applied to software that is not dependent on the properties of a particular machine, and can therefore be used on any machine. Such software is also described as portable.
High-level programming language Python, JavaScript, Visual Basic, Delphi, Perl, PHP, ECMAScript, Ruby, C#, Java and many others.
Object-oriented programming (OOP) a style of programming characterized by the identification of classes of objects closely linked with the methods (functions) with which they are associated..
Scripting language a type of programming language that is interpreted rather than requiring compilation.
Data types:
  • Integer- ( int ): represents positive or negative whole numbers like 3 or -512. A datum of integral data type
  • Float - Floating point number ( float ): represents real numbers like 3.14159 or -2.5.
  • String - Character string (usually called “string”, str ): text.
    • Primitive data type a set of basic data types from which all other data types are constructed.
    Data structure a specialized format for organizing, processing, retrieving and storing data.
    Python collection data structures containers for different kinds of data. The four main types are lists, sets, tuples and dictionaries.
    Python List Lists are used to store multiple items in a single variable.
    Python Tuples Tuples are used to store multiple items in a single variable. Tuple is one of 4 built-in data types in Python used to store collections of data, the other 3 are List, Set, and Dictionary, all with different qualities and usage.
    Python Sets Sets are used to store multiple items in a single variable. Set is one of 4 built-in data types in Python used to store collections of data, the other 3 are List, Tuple, and Dictionary, all with different qualities and usage.
    Python Dictionaries Dictionaries are used to store data values in key:value pairs.
    Global Variables Variables that are created outside of a function (as in all of the examples above) are known as global variables. Global variables can be used by everyone, both inside of functions and outside.
